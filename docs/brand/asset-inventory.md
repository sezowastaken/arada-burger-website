# Asset Inventory

This file tracks the location and naming of all brand and product assets.

## Directory Structure
- `frontend/public/brand/`: Logos, mascots, and UI textures.
- `frontend/public/menu/pdf/`: Downloadable menu files.
- `frontend/public/menu/products/`: Category-specific product images.

## Naming Conventions
- Use lowercase with hyphens: `classic-burger.jpg`.
- Be descriptive: `logo-main-color.png`, `mascot-happy-burger.svg`.
- Images should be optimized for web performance before being added.
